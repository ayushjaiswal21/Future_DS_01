# FUTURE_DS_01 - Business Sales Dashboard from E-Commerce Data 📊

This repository contains the first task for the **Future Interns Data Science & Analytics Internship**, focused on analyzing e-commerce data and delivering actionable business insights through an interactive dashboard built using **Jupyter Notebook** and **Plotly**.

---

## 🧠 Task Objective
To analyze e-commerce sales data to:
- Identify best-selling products
- Monitor monthly sales trends
- Discover high-revenue product categories and regions
- Evaluate returned orders and manager-wise regional structure

---

## 🧰 Tools & Technologies Used
- **Python**
- **Jupyter Notebook**
- **pandas, numpy** – data processing
- **plotly.express** – interactive visualizations
- **openpyxl** – Excel file handling
- **nbconvert** – notebook to HTML export

---

## 📊 Dashboard Highlights

- **Top 10 Best-Selling Products**: Interactive bar chart showing leading products by revenue.
- **Monthly Sales Trend**: Line chart to track business seasonality.
- **Sales by Category and Sub-Category**: Visual breakdowns for focused business strategy.
- **Regional Sales Distribution**: Identify high-performing regions.

## 🚀 How to Run
1. Clone the repo
2. Place dataset in `Dataset/`
3. Open the notebook from `Notebook/sales_dashboard.ipynb`
4. Run all cells to generate visuals and KPIs
5. Export to HTML using:
```bash
jupyter nbconvert --to html sales_dashboard.ipynb
````
